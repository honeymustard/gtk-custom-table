## Gtk-Custom-Table

A custom table implementation for Gtk+3 written in C.

## Compilation instructions

**Linux:**

    $ make
    
**Windows:**

    $ mingw32-make mingw32-make

    # builds gtk_custom_table.dll gtk_custom_table.dll.a
