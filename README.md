## Gtk-Custom-Table

A custom table implementation for Gtk+3 written in C.

## Compilation instructions

**Linux:**

    $ make                         # builds gtk_custom_table.so
    
**Windows:**

    $ mingw32-make mingw32-make    # builds gtk_custom_table.dll
